Overview of Features

The Patrick theme is a very flexible theme, allowing you to create something unique for your site. Highlights of some of the most notable features:

  • Integration with Pictaculous to create a unique color palette. Within Theme Options > Color Palette, supply an image URL to the Source Image tab of the Pictaculous Palette field. Select an extracted palette then assign palette colors to named locations of the theme, creating a unique look for your blog.
  • Further individualize your site with fonts from the Google Web Fonts directory. In Theme Options > Design you can select a different typeface (the font name), variation (regular, bold, italic, bold italic), and size for the body text and headings.
  • A variety of design options to help individualize your blog: one, two, and three column layouts; control element transparency; Entry and Page meta display position; and home page carousel (with the jQuery Cycle plugin).
  • Facebook and Twitter integration. Facebook Commenting, Like button, and Recommendations social plugins along with Open Graph meta are included. The Twitter retweet button is also included.

Theme Options

Each Theme Option field is not detailed here; the hint of the field should (hopefully) offer adequate documentation of the feature. Some features likely require a little extra explanation, and that's what you'll find here.

Callbacks

“Callback” areas are provided for you to further customize Patrick with any HTML or Javascript, for example. You'll find some callback options in the Blog Callbacks tab, but there are also a few in other tabs: Blog Header, Entry Formatting, and Page Formatting.

Color Palette

The Pictaculous Palette field returns eleven palettes from the image URL you supply, found in the Choose Palette tab. Five are from Adobe Kuler, five are from COLOURlovers, and one is from Pictaculous. I don't have any idea how Pictaculous determines what colors to extract from your image to create the palette. Some of the results are beyond dubious (where all five colors in the palette are the same, even). If you're not happy with the results, try cropping the image differently or try a different image altogether.

When the same color is used in some adjacent named locations, spacing may collapse and therefore disappear. For example, if the Widget Background and Beta/Gamma Background are set to the same color, the spacing between them will collapse to disappear, maintaining the expected amount of padding around the element. This is a vague description, I know. Most can ignore this note because the spacing will just work and look good, but if you're using some custom CSS, this may be important to understand.

Design

Setting a large radius in the Layout > Border Radius Theme Option (say, larger than about 10) can have undesirable side effects: spacing around the Facebook Recommendations plugin, for example, has a square overlap. Depending upon the other options and widgets you use, you may want to avoid a large radius (if you choose to use a radius at all).

Home Page Formatting

The Entry Display Configuration section controls how entries are displayed on the home page. Entries are displayed in the following order:

  1. Featured entries: display with a large image as the background, as carousel that cycles through the most recent entries.
  2. Summarized entries: display in an "excerpt" format including the entry title (linked), meta, and excerpt text.
  3. Listed entries: display as a bulleted list of linked entry titles.

The section Featured Entries Carousel Configuration allows you to configure the Carousel, which uses the jQuery Cycle plugin. The variety of Display Effects and speeds directly configure the plugin. I tried to create these options in a way that would require no knowledge of jQuery or the Cycle plugin, however to understand all of the effects you'll want to refer to the Cycle documentation.

Facebook

There are a variety of features in the Patrick theme to integrate with Facebook. In order to moderate comments and view analytics you'll need to enter a Facebook Application ID. Creating a Facebook application is easy: just follow Facebook's steps.

The Facebook Like button width can be configured, and the height is controlled by other options. Setting the width, in particular, is likely just a matter of trial and error to make everything fit as you like.

Twitter

The Twitter username is required for the Follow button, but optional for use with the Retweet button.

Adding Additional Widgets

Additional widgets can be created for use in the sidebars. In order to make their formatting match other widgets, they will need to follow some basic formatting provided by some simple HTML:

<div class="widget-archive-author widget-archive widget">
    <h3 class="widget-header">Widget Title</h3>
    <div class="widget-content">
        <p>Any widget content goes here.</p>
    </div>
</div>

The widget header line is optional; if you don't want to include a title with your widget just exclude the following:

<h3 class="widget-header">Widget Title</h3>

Content should be added in the "widget-content" element, as you might guess.

Custom CSS

Want to make some style customizations to this theme? The best way to do that is with the Custom CSS plugin. The Custom CSS plugin provides an editing interface to add your own styles.
